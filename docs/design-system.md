\# TechQuest Design System



\## Design Philosophy



TechQuest should feel:



\- Welcoming

\- Modern

\- Calm

\- Motivating

\- Inclusive



The interface should encourage curiosity rather than make users feel like they are taking an exam.

\# Spacing



Use an 8px spacing system.



Examples:



8px

16px

24px

32px

48px

64px



\---



\# Typography



Headings:

Bold



Body:

Easy to read



The interface should prioritize readability over decoration.



\---



\# Design Principles



1\. Simplicity first.

2\. Reward curiosity.

3\. Celebrate progress.

4\. Keep interfaces clean.

5\. Every screen should feel welcoming.
